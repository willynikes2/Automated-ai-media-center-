# Setup Guide

## Prerequisites

- **Docker Engine 24+** — [Install Docker](https://docs.docker.com/engine/install/)
- **Docker Compose v2** — Included with modern Docker Desktop
- **Domain** (optional) — For HTTPS via Let's Encrypt
- **TMDB API Key** (free) — [Get one here](https://www.themoviedb.org/settings/api)
- **Real-Debrid Token** (optional) — [Get token](https://real-debrid.com/apitoken)
- **VPN Credentials** (optional) — For torrent acquisition fallback

## First Run

### 1. Clone and navigate

```bash
cd invisible-arr/edge-node
```

### 2. Run the installer

```bash
bash scripts/install.sh
```

The installer will:
1. Verify Docker and Docker Compose are installed
2. Copy `.env.template` to `.env` and generate a random Postgres password
3. Prompt for your TMDB API key, Real-Debrid token, domain, VPN settings
4. Create required data directories
5. Build and start all containers

### 3. Verify

```bash
bash scripts/smoke.sh
```

### 4. Configure Seerr

1. Open Seerr at `http://localhost:5055`
2. Connect to Jellyfin (http://jellyfin:8096)
3. Add Sonarr (http://sonarr:8989) and Radarr (http://radarr:7878)
4. Configure Prowlarr indexers at http://localhost:9696 (internal only)

### 5. Configure Prowlarr Indexers

1. Access Prowlarr via `docker compose exec` or internal port
2. Add your preferred indexers
3. For Cloudflare-protected indexers, set FlareSolverr as proxy (see [FLARESOLVERR.md](FLARESOLVERR.md))

## Environment Variables Reference

| Variable | Default | Description |
|----------|---------|-------------|
| `DOMAIN` | *(empty)* | Public domain for Traefik TLS. Leave empty for HTTP-only |
| `ACME_EMAIL` | *(empty)* | Let's Encrypt email for TLS certificates |
| `TMDB_API_KEY` | *(required)* | The Movie Database API key |
| `RD_API_TOKEN` | *(optional)* | Real-Debrid API token |
| `RD_ENABLED` | `false` | Enable Real-Debrid acquisition path |
| `VPN_ENABLED` | `false` | Enable Gluetun VPN for torrent traffic |
| `VPN_PROVIDER` | *(empty)* | Gluetun provider (mullvad, nordvpn, protonvpn) |
| `VPN_TYPE` | `wireguard` | VPN protocol type |
| `LLM_PROVIDER` | `none` | Optional LLM for tie-breaking (none/openai/anthropic) |
| `DATA_PATH` | `./data` | Root path for media and downloads |
| `CONFIG_PATH` | `./config` | Configuration directory for services |
| `MEDIA_PATH` | `./data/media` | Media library path |
| `DOWNLOADS_PATH` | `./data/downloads` | Downloads staging path |
| `POSTGRES_USER` | `invisiblearr` | Database username |
| `POSTGRES_PASSWORD` | *(generated)* | Database password (auto-generated by install.sh) |
| `DEFAULT_MAX_RESOLUTION` | `1080` | Default quality cap for new users |
| `DEFAULT_ALLOW_4K` | `false` | Allow 4K downloads by default |
| `IPTV_ENABLED` | `false` | Enable IPTV gateway service |
| `PUID` | `1000` | User ID for file permissions |
| `PGID` | `1000` | Group ID for file permissions |
| `TZ` | `America/New_York` | Timezone |
